# svg-animation-to-png
svg animation to png files, covert to video

> 针对SMIL的动画svg文件，需转成png图片由底层去合成视频

svg 必须配置一下属性

* width
* height
* animate dur

固定模板，保持dur一直

### 体验

1. [下载](https://raw.githubusercontent.com/tomieric/svg-animation-to-png/master/animate.svg) `animate.svg`
2. 在 [https://tomieric.github.io/svg-animation-to-png/](https://tomieric.github.io/svg-animation-to-png/) 上传svg文件
3. 只支持chrome
